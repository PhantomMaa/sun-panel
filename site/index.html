快速部署

只需几个简单步骤，即可开始使用 MoonBox

01

拉取 Docker 镜像

使用 Docker 命令拉取最新的 MoonBox 镜像。

docker pull ghcr.io/phantommaa/sun-panel/monolith:latest
02

启动容器

配置端口映射和数据卷，启动 MoonBox 容器。

docker run -d --name moonbox -p 8085:8085 -v /your/data/path:/app/data ghcr.io/phantommaa/sun-panel/monolith:latest
03

访问面板

通过浏览器访问 http://localhost:8085，开始使用 MoonBox。
